# Multiverso News - Site de notícias Geek
Este projeto é uma página de notícias fictícias, feito com tecnologias básicas de front-end, contendo um carrossel de personagens geek e páginas 100% personalizadas.

# Demonstração
![Demontração do projeto](Multiverso_News/multiversoNewsDemonstracao.gif)

## Tecnologias utilizadas

- HTML5;
- CSS3;
- JavaScript;

## Funcionalidades

- [X] Carrossel de personagens geek com movimento horizontal automático;
- [X] Páginas com notícias separadas por categorias;
- [X] Páginas específicas de cada notícia;
